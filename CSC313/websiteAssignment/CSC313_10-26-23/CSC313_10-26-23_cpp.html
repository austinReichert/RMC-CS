C++

A low-level language that operates with Object Oriented Programming.

What is C++?

"C++ is a general-purpose, free-form programming language created by Bjarne Stroustrup ..., as an enhancement to the C language. Since it is an extension of C, it is also known as C with classes. It was specifically designed with an orientation towards large systems and resource-constrained software... The biggest advantage of C++ is that it is super scalable and allows developers to have a lot of control over how their applications use up resources." [1] C++ is considered a low-level language due to the lack of automatic memory management.[2]

How does C++ work?

C++, simply put, works by using a compilers to take the code and convert it into an object file. From there the compiler passes the processed object file to a linker and produces an executable file. [3]

Example codes

C++, along with many other languages, requires "boilerplate code". Boilerplate code is code that simply allows your program to run.

C++ Boilerplate 
#include <iostream>

int main() {
    // stuff here
    return 0;
}

Utilizing this boilerplate code we can add functionality by putting code within the main function.

Example code for "Hello world!" 
#include <iostream>

int main() {
    std::cout << "Hello world!";
    return 0;
}
Output: 
Hello world!

Furthermore, using multiple classes we can create objects with methods and attributes.

Example code for an object with methods and attributes. 
#include <iostream>

class Dog {
    public:
        void setName(std::string newName){
            this->name = newName;
        }
        std::string getName(){
            return this->name;
        }
        void setAge(int newAge){
            this->age = newAge;
        }
        int getAge(){
            return this->age;
        }
    private:
        std::string name;
        int age;
};

int main() {
    Dog myDog;
    myDog.setName("Harold");
    myDog.setAge(5);
    std::cout << "My dogs name is " << myDog.getName() << ".\n";
    std::cout << "My dogs age is " << myDog.getAge() << ".\n";
    return 0;
}
Output: 
My dogs name is Harold.
My dogs age is 5.
