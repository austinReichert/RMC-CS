Object Oriented Programming

Simple, widely used programming paradigm.

What is Object Oriented Programming?

According to techtarget.com object oriented programming is "...a computer programming model that organizes software design around data, or objects, rather than functions and logic. An object can be defined as a data field that has unique attributes and behavior. OOP focuses on the objects that developers want to manipulate rather than the logic required to manipulate them. This approach to programming is well-suited for programs that are large, complex and actively updated or maintained. This includes programs for manufacturing and design, as well as mobile applications; for example, OOP can be used for manufacturing system simulation software." [1] These programs often use classes, methods, objects (obviously) and attributes as building blocks. [1]

Who uses it?

As explained on educative.io, "Object-oriented programming (OOP) is a fundamental programming paradigm used by nearly every developer..."[2]. A few of the contributing factors to its wide use is its flexibility, reusablity[1] and comprehensible structure.
